# Requirements
Python 3.8+

# Local development

### Set up the project
```sh
cd ~/YOUR_PATH/python-worker-public
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Run tests
```sh
venv/bin/python3 -m pytest tests
```

### Run code linter
```sh
venv/bin/python3 -m flake8
```
